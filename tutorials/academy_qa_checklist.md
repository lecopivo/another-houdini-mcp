# Academy QA Checklist

Run this before marking any node as `studied`.

## Per-Node Gate

- [ ] Documentation reviewed (`help/nodes/<context>/<node>.txt`)
- [ ] Example notes/files reviewed (`help/examples/nodes/<context>/<node>/`)
- [ ] OTL loaded, instantiated, and unlocked for inspection
- [ ] Internal graph inspected (connections + key parameters)
- [ ] Node comments read
- [ ] Sticky notes/post-it notes read
- [ ] At least one behavior observation recorded from live scene output
- [ ] Intent vs observed distinction documented
- [ ] Node note updated
- [ ] Context and academy progress updated
